3.7.4  omp_get_num_devices

Summary The omp_get_num_devices routine returns the number of non-host devices available for offloading code or data.

Format

 

 
int omp_get_num_devices(void);
  

 

 
integer function omp_get_num_devices()
  

Binding The binding task set for an omp_get_num_devices region is the generating task.

Effect The omp_get_num_devices routine returns the number of available non-host devices onto which code or data may be offloaded. When called from within a target region the effect of this routine is unspecified.
