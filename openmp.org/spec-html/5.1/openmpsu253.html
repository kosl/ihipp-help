5.6.5  Beginning OpenMP Threads

Summary When starting an OpenMP thread, the implementation executes ompd_bp_thread_begin.

Format

 

 
void ompd_bp_thread_begin(void);
  

Description The OpenMP implementation must execute ompd_bp_thread_begin at every native-thread-begin and initial-thread-begin event. This execution occurs before the thread starts the execution of any OpenMP region.
