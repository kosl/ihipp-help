3.7.3  omp_get_default_device

Summary The omp_get_default_device routine returns the default target device.

Format

 

 
int omp_get_default_device(void);
  

 

 
integer function omp_get_default_device()
  

Binding The binding task set for an omp_get_default_device region is the generating task.

Effect The omp_get_default_device routine returns the value of the default-device-var ICV of the current task. When called from within a target region the effect of this routine is unspecified.
