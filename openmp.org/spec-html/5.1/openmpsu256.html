5.6.8  Finalizing OpenMP Devices

Summary When terminating an OpenMP thread, the implementation executes ompd_bp_device_end.

Format

 

 
void ompd_bp_device_end(void);
  

Description The OpenMP implementation must execute ompd_bp_device_end at every device-finalize event. This execution occurs after the thread executes all OpenMP regions. After execution of ompd_bp_device_end, any address_space_handle that was acquired for this device is invalid and should be released.
